용도 : JBTS Project용
언어 : HTML, PHP, Java Script, CSS

저는 이 홈페이지를 Debian에 있는 Apache를 이용했습니다.

Linux를 사용할 줄 모르는 사람이라도 PHP 및 HTML을 실행 시킬 서버를 사용 할 줄 알면 상관없습니다.

기본적으론 BOOTSTRAPMADE가 배포한 HTML을 기본으로 합니다. (그래서 몇몇 Java Script, CSS 파일들은 설명이 영어거나 생략되어있습니다.)

PHP는 다른 기능보단 Mysql 접속을 위해 사용했습니다. (PHP는 깔끔하지 않을수도 있습니다.)

Java Script는 jquey를 주로 사용했습니다. ( 이미지 슬라이드 등 )

CSS는 기본 스킨은 유지한채 Font Awesome(icon)과 Roll Over기능 및 기타 Style 작업을 위해 사용했습니다.

* 몇몇 라이브러리는 인터넷이 사용가능해야합니다.
* 어느정도 기본 지식을 바탕으로 봐야합니다.
* 각 조별로 겹치는 설정이 많기때문에 조금은 생략되 있을 수 있습니다.
* custom.css/ mouse.js 외의 파일은 기본 스킨과 jquery 및 처음 받아온 사이트의 기능입니다.
